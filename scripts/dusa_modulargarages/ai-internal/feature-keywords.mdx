## Search Patterns

### For Code Changes

| What to Change | Search Pattern | Files to Modify |
|----------------|----------------|-----------------|
| Add new callback | `lib.callback.register` | server/api/sv_api.lua |
| Add new NUI callback | `RegisterNUICallback` | client/core/interaction.lua, client/main.lua |
| Add new event | `RegisterNetEvent` | client/main.lua, client/core/*.lua |
| Add new config option | `Config.` | config/shared.lua |
| Add new database table | `CREATE TABLE` | sql/release/consolidated_schema.sql |
| Add new locale key | JSON key | locales/*.json |

### For Debugging

| Issue Area | Search Pattern | Files to Check |
|------------|----------------|----------------|
| Vehicle spawn | `spawnVehicle`, `CreateVehicle` | sv_api.lua, interaction.lua |
| Database | `MySQL.query`, `MySQL.insert` | database.lua, sv_api.lua |
| Permissions | `checkPermission`, `ValidateOwnership` | validation.lua, vehicle_ownership.lua |
| UI state | `SendNUIMessage`, `SetNuiFocus` | interaction.lua, main.lua |
| Zone detection | `ox_target`, `addBoxZone` | interaction.lua |
