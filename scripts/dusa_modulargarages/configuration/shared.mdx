# Shared Configuration

The `config/shared.lua` file contains settings used by both client and server scripts.

## Locale

```lua
Config.Locale = "en"  -- Available: en, tr, de, es, fr, it, pl
```

Supported languages are stored in `locales/*.json`.

## Default Settings

```lua
Config.DefaultSettings = {
    MaxVehicles = 50,  -- Maximum vehicles per garage (display)
}
```

## Vehicle Transfer Pricing

Configure the vehicle transfer system:

```lua
Config.Prices.vehicleTransfer = {
    enabled = true,              -- Enable/disable transfers
    feeType = "flat",            -- "flat" or "distance"
    flatFee = 350,               -- Flat fee amount
    accountType = "bank",        -- "bank" or "cash"
    
    -- NULL garage handling
    allowTransferFromNullGarage = true,
    allowTransferIfSpawned = true,
    nullGarageRetrievalFee = 250,
    
    -- Distance-based pricing
    distanceMultiplier = 0.5,    -- Cost per meter
    minTransferFee = 100,        -- Minimum fee
    maxTransferFee = 50000,      -- Maximum fee cap
}
```

### Fee Calculation

| Fee Type | Calculation |
|----------|-------------|
| `flat` | Fixed `flatFee` amount |
| `distance` | `distance * distanceMultiplier`, clamped to min/max |

## Repair Pricing

Part-based vehicle repair pricing:

```lua
Config.Prices.repair = {
    enabled = true,
    accountType = "bank",
    pricingMethod = "percentage",  -- "fixed" or "percentage"
    fullRepairDiscount = 0.0,      -- Discount for full repair
    
    parts = {
        engine = {
            cost = 500,
            bone = "engine",
            displayName = "Engine"
        },
        tires = {
            cost = 300,
            bones = {"wheel_lf", "wheel_rf", "wheel_lr", "wheel_rr"},
            displayName = "Tires"
        },
        body = {
            cost = 400,
            bone = "bodyshell",
            displayName = "Car Body"
        }
    }
}
```

### Pricing Methods

| Method | Calculation |
|--------|-------------|
| `fixed` | `parts[part].cost` regardless of damage |
| `percentage` | `damagePercent * parts[part].cost` |

## Mechanic Integration

Lock vehicles under active mechanic work orders:

```lua
Config.MechanicIntegration = {
    Enabled = true,
    ResourceName = "dusa_mechanicv2",
    LockOnOrderAccepted = true,
    
    LockingStatuses = {
        "in_progress",
    },
}
```

```lua
        displayName = "string",
        minGrade = "number",
        category = "string",
        description = "string"
    },
    Constraints = {
        minGrade = { min = 0, max = 10 }
    }
}
```
