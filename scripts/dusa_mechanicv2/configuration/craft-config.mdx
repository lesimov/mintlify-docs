# Craft Yapılandırması

`shared/craft_constants.lua` dosyası sahte plaka ve nitro craft sisteminin sabitlerini içerir.

## Tarif ID'leri

```lua
CraftConstants.RECIPE_IDS = {
    FAKE_PLATE = 'fake_plate',
    FAKE_PLATE_REMOVER = 'fake_plate_remover',
    NITRO_CRAFT = 'nitro_craft',
    NITRO_REFILL = 'nitro_refill',
}
```

## Item Tipleri

```lua
CraftConstants.ITEM_TYPES = {
    FAKE_PLATE = 'fake_plate',
    FAKE_PLATE_REMOVER = 'fake_plate_remover',
    NITRO = 'nitro',
    NITRO_REFILL = 'nitro_refill',
}
```

## Çıktı Item'ları

```lua
CraftConstants.OUTPUT_ITEMS = {
    FAKE_PLATE = 'fake_plate',
    FAKE_PLATE_REMOVER = 'fake_plate_remover',
    NITRO_FILLED = 'nitro_filled',
    NITRO_EMPTY = 'nitro_empty',
}
```

## Malzemeler

```lua
CraftConstants.MATERIALS = {
    -- Sahte Plaka
    BLANK_PLATE = 'blank_plate',
    PLATE_INK = 'plate_ink',
    -- Sahte Plaka Sökücü
    SCREWDRIVER = 'screwdriver',
    SOLVENT = 'solvent',
    -- Nitro
    NITRO_BOTTLE = 'nitro_bottle',
    NITRO_KIT = 'nitro_kit',
    NITRO_EMPTY = 'nitro_empty',
}
```

## Heat (Isı) Değerleri

```lua
CraftConstants.HEAT = {
    FAKE_PLATE = 3,      -- Plaka takıldığında
    NITRO_PER_USE = 5,   -- Nitro kullanımı başına
}
```

## Craft Süreleri

```lua
CraftConstants.CRAFT_TIMES = {
    FAKE_PLATE = 15,         -- 15 saniye
    FAKE_PLATE_REMOVER = 10, -- 10 saniye
    NITRO_CRAFT = 20,        -- 20 saniye
    NITRO_REFILL = 10,       -- 10 saniye
}
```

## Nitro Ayarları

```lua
CraftConstants.NITRO = {
    DEFAULT_KEY = 'G',
    POWER_MULTIPLIER = 2.0,   -- 2x güç
    DURATION = 3.0,           -- 3 saniye boost
    DRAIN_RATE = 1,           -- Aktivasyon başına tüketim
    COOLDOWN = 5.0,           -- 5 saniye bekleme
    SCREEN_EFFECTS = true,
    REAR_LIGHT_TRAILS = true,
    PARTICLE_SCALE = 1.0,
    SOUND_ENABLED = true,
    ALLOW_IN_WATER = false,
    REQUIRE_THROTTLE = true,
    ELECTRIC_VEHICLES = {     -- Nitro yasak araçlar
        'neon', 'voltic', 'cyclone', 'tezeract', ...
    },
}
```
